# API Overview

This document captures a starting point for API documentation. It outlines
primary resources and the endpoints that should be expanded with request/response
examples and edge cases as we continue building.

## Invoices
- `GET /api/invoices`
- `GET /api/invoices/:id`
- `POST /api/invoices`
- `PUT /api/invoices/:id`
- `DELETE /api/invoices/:id`

## Campaigns

- `GET /api/campaigns/:id`
- `POST /api/campaigns`
- `PUT /api/campaigns/:id`
